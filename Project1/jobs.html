Cloud Engineer & Cybersecurity Engineer - Job Descriptions

Detailed job roles, qualifications, and application information for both Cloud Engineering and Cybersecurity positions.

Cloud Engineer (Infrastructure & Automation)

Position Overview

Title: Cloud Engineer

Reference Number: A0001

Brief Description: Design, build, and operate scalable cloud infrastructure and automation pipelines to support production services and developer platforms.

Salary Range: USD 80,000 – 120,000 per year

Reports To: Lead Infrastructure Engineer

Key Responsibilities

  • Design and implement cloud architectures (IaaS/PaaS).
  • Develop infrastructure-as-code using Terraform/CloudFormation.
  • Maintain CI/CD pipelines for deployment and testing.
  • Monitor system performance and handle incidents.
  • Collaborate with development teams to ensure reliability and security.

Required Qualifications, Skills & Attributes

Essential
  • Bachelor's degree in Computer Science, Engineering, or related field.
  • 3+ years managing cloud infrastructure (AWS, Azure, or GCP).
  • Proficiency in Terraform or CloudFormation.
  • Experience with scripting (Python, Bash) and Git.
  • Knowledge of Docker/Kubernetes.
Preferable
  • Certifications: AWS Solutions Architect, Azure Administrator, or GCP Professional.
  • Experience with cost optimization and advanced networking.
  • Familiarity with policy-as-code tools (OPA, Sentinel).

Specific Tasks

  1. Create and maintain Terraform modules for new services.
  2. Build blue/green deployment pipelines using GitOps.
  3. Respond to alerts and conduct post-incident analysis.
  4. Optimize cloud resource usage and spending.

Cybersecurity Engineer (Defence & Detection)

Position Overview

Title: Cybersecurity Engineer

Reference Number: A0002

Brief Description: Protect systems and data by designing detection, prevention, and response capabilities; perform vulnerability assessments and implement security controls across environments.

Salary Range: USD 75,000 – 115,000 per year

Reports To: Security Operations Manager

Key Responsibilities

  • Develop and tune IDS/IPS, EDR, and SIEM rules for early threat detection.
  • Conduct vulnerability scans and coordinate penetration testing.
  • Implement and maintain security controls for IAM and cloud security.
  • Lead incident response and create documentation.

Required Qualifications, Skills & Attributes

Essential
  • Bachelor’s degree in Cybersecurity, Computer Science, or related field.
  • 2+ years of experience in security operations or vulnerability management.
  • Experience with SIEM tools (Splunk, Elastic) and endpoint protection.
  • Strong understanding of operating systems, networking, and scripting (Python/PowerShell).
Preferable
  • Certifications: CISSP, OSCP, CEH, or cloud security credentials (CCSP, AWS Security).
  • Experience integrating security into CI/CD pipelines.
  • Knowledge of compliance standards like ISO27001 or SOC2.

Specific Tasks

  1. Investigate and triage security alerts and document findings.
  2. Run regular vulnerability scans and oversee remediation.
  3. Create detection rules and automated incident response scripts.

Candidate Notes & Application Tips

Benefits: Flexible hours, hybrid work, certification reimbursement, and training programs.

How to Apply:attach your CV with a short cover letter.

  • Focus on measurable achievements in previous roles.
  • Provide project links or examples of automation/security improvements.

Recruitment & Selection Process

The standard recruitment process includes:

  1. Initial application and CV screening.
  2. Technical assessment (relevant to the position applied).
  3. Interview with technical and management leads.
  4. Final HR and offer stage.
